Data Cleaning

To test K-Means Clustering we have follow its input data requirements:
* Numerical variables only
* Data must have no noises or outliers
* Data has symmetric distribution of variables (it isnâ€™t skewed)
* Variables on the same scale
* There is not collinearity
* Few numbers of dimensions (Not necessary but as a general rule of thumb; fewer the better)

Data Processing

As mentioned in-order for us to utilize the K-Means algorithm the first rule is we must have numerical variables however our data-set is a mix of both numerical and categorical variables

So how are going to process it in a way where weâ€™re able to use K-Means Clustering despite the majority of the variables being categorical?

There are multiple algorithms that we can achieve clustering despite the data-set being highly categorical; like K-modes, DP-means etcâ€¦ We can still use K-means but use a different measure of distance; (instead of Euclidean) Gower Distance.

Aside from K-Means and Gower we can still utilize K-Means Clustering despite having a mix of Numerical and Categorical and that is to map ordinal data into numerical values! Interesting isnâ€™t it? However this is not a good idea if your chosen categorical variables have no natural order but as observed in the PISA data-set most categorical variables are in a sense ordinal, so weâ€™re gonna utilize only those categorical variables that will make sense if ordered.

See here: K-means Clustering on Ordinal Data

Hereâ€™s the list of variables that is viable for mapping:

  • Fatherâ€™s education
  • Motherâ€™s education
  • Number of Computers the Student have
  • Number of Cars the Studentâ€™s family have
  • Number of TVâ€™s the Studentâ€™s family have
  • Number of Bookâ€™s the Studentâ€™s have read

How about the already existing numerical values? Weâ€™re gonna leave them as is and scale/normalize them later on.

Iâ€™m gonna map the aforementioned variables above in a scale of 0 to 10 Before we proceed to mapping weâ€™re gonna have to convert all variables with factor types to character for us to reassign mapped values
